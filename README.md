# mikirystudios.com

Personal site for Daniel Cedeño (mikiry) — portfolio, and home of mkrHub.

## Stack

- [Next.js](https://nextjs.org/) (App Router, TypeScript)
- [Tailwind CSS](https://tailwindcss.com/)
- Hosting: Vercel (free tier), custom domain `mikirystudios.com`
- Planned: Supabase (db/auth/storage), Stripe (checkout), Resend (transactional email), Mailchimp (newsletter)

## Structure

- `/` — landing
- `/portfolio` — exhibition-style project gallery + credits
- `/mkrhub` — mkrHub download (pay-what-you-want, $0 floor)
- `/about` — bio

## Status

Phase 1 scaffold. Portfolio and mkrHub content are placeholders pending final
assets and payment/email backend wiring.

## Development

```bash
npm install
npm run dev
```
